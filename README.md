# ▶️ Installation

npm install

---

# ▶️ Run Tests

## Run All Tests

npx playwright test

## Run Specific Test File

npx playwright test tests/ui/spec/TestWithPOM.spec.ts

## Headed Mode

npx playwright test --headed

## Debug Mode

npx playwright test --headed --debug

---

# 📊 Show Report

npx playwright show-report
